Indian E-commerce Sales Dashboard (Power BI)


📌 Overview

This project presents an interactive Power BI dashboard built to analyze Indian e-commerce sales data. It provides insights into sales performance, customer behavior, and product trends to support data-driven decision-making.


📊 Features
📈 Sales and Profit Analysis
🛍️ Product Category Insights
🌍 Region-wise Performance
👥 Customer Segmentation
📅 Time-based Trends (Monthly/Yearly)
📌 Key Performance Indicators (KPIs)


🧰 Tools & Technologies
Power BI – Data visualization and dashboard creation
Microsoft Excel / CSV – Data source (assumed)
Data Cleaning & Transformation – Power Query


📂 Project Structure
Indian E-commerce.pbix   # Main Power BI dashboard file
README.md                # Project documentation
